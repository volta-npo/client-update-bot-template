# Client Update Bot Template v1.0 Product Specification

## Workbench

Weekly Client Update Builder

## Domain table

Project status items

## Metrics

- Update Completeness
- On-time Cadence
- Risk Clarity

## Domain rows

- Milestones entered
- Completed work summarized
- Blockers logged
- Owner asks listed
- Risks rated
- Next steps written
- Mentor notes reviewed
- Weekly update exported

## Generated deliverables

- Email update
- Slack-style update
- Timeline CSV

## Validation gates

- Next step and owner ask required
- Risk level required
- Reviewer required before sending
